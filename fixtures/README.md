# Test Fixtures

The project is a **Go package**; Python is used only to generate these test cases. This directory contains test fixtures generated by the Python scripts in `../scripts/`.

Fixtures are generated files and should not be committed to version control.

## ECDSA Fixtures

Generated by `scripts/flawed_signer.py`:
- `test_signatures_same_nonce.json` - Nonce reuse (k2 = k1)
- `test_signatures_counter.json` - Counter-based (k2 = k1 + 1)
- `test_signatures_affine.json` - Affine relationship (k2 = 2*k1 + 1)
- `test_signatures_hardcoded_step.json` - Hardcoded step (k2 = k1 + 12345)
- `test_key_info.json` - Private/public key information

## EdDSA Fixtures

Generated by `scripts/flawed_eddsa_signer.py`:
- `test_eddsa_signatures_same_nonce.json` - Nonce reuse (r2 = r1)
- `test_eddsa_signatures_counter.json` - Counter-based (r2 = r1 + 1)
- `test_eddsa_signatures_affine.json` - Affine relationship (r2 = 2*r1 + 1)
- `test_eddsa_signatures_hardcoded_step.json` - Hardcoded step (r2 = r1 + 12345)
- `test_eddsa_key_info.json` - Private/public key information

**Note:** EdDSA fixtures simulate flawed implementations using random nonces (non-standard).
